# MCP-llama.cpp-demo
Build a fully local AI agent using:   
  llama.cpp  Qwen2.5-1.5B-Instruct (GGUF) 
  Model Context Protocol (MCP) 
  Python  
No OpenAI API key. No cloud inference. Everything runs locally.
